FAQ

How do I connect to a server?
Tap the bolt icon for Quick Connect, or add a saved host with the + button. Enter your server's hostname, username, and password or SSH key.

Where are my passwords and keys stored?
Only in the iOS Keychain on your device, protected by device encryption and (optionally) Face ID. They are never uploaded anywhere.

What is the fingerprint prompt on first connect?
That is Trust-On-First-Use (TOFU) host-key verification, which protects you from man-in-the-middle attacks. Approve it only if you trust the server.

Which keys are supported?
Ed25519 and RSA. Generate an Ed25519 key on device, or import your existing OpenSSH keys.
